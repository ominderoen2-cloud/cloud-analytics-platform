# Cloud Analytics Platform (IaC-Driven)

This project demonstrates how to provision cloud infrastructure using Terraform,
ingest raw data into a cloud data warehouse, transform it using dbt, and present
analytics via a simple web dashboard.

## Tech Stack
- Google Cloud Platform (GCP)
- Terraform (Infrastructure as Code)
- BigQuery
- dbt
- SQL
- HTML / CSS

## Goal
Build an end-to-end cloud analytics system that reflects real-world DevOps and
analytics engineering workflows.
